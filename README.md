```
└── README.md          # 이 파일
```

## 🚀 GitHub Pages 배포 방법

1. GitHub에 새 저장소(Repository) 생성
2. 이 폴더의 모든 파일 업로드
3. Settings → Pages → Source에서 `main` 브랜치 선택
4. 저장 후 몇 분 뒤 `https://[username].github.io/[repository-name]` 에서 확인

## ⚠️ 주의사항

- `images` 폴더에 모든 멘토 이미지 파일을 넣어주세요
- 이미지 파일명이 정확히 일치해야 합니다
